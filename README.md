# Portfolio Profesional - Álvaro Alonso Fernández

Portfolio personal de alto rendimiento desarrollado con **React**, **TypeScript** y **Tailwind CSS**. Este proyecto destaca por su diseño minimalista, animaciones fluidas y una integración dinámica con la API de GitHub.

## 🚀 Características Principales

- **Integración Dinámica con GitHub**: Los proyectos se cargan en tiempo real consumiendo la API de GitHub, mostrando estrellas, forks y lenguajes actualizados automáticamente.
- **Diseño Responsivo y Moderno**: Interfaz optimizada para todos los dispositivos siguiendo principios de diseño limpio y profesional.
- **Currículum Vitae Integrado**: Recreación visual de alta fidelidad del CV original para una visualización directa sin necesidad de visores externos.
- **Sección de Aprendizaje Continuo**: Espacio dedicado a mostrar las tecnologías y arquitecturas en las que estoy profundizando actualmente.
- **Animaciones Fluidas**: Uso de `motion/react` para transiciones suaves y una experiencia de usuario premium.

## 🛠️ Stack Tecnológico

- **Frontend**: React 18+ con Vite
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS
- **Iconografía**: Lucide React
- **Animaciones**: Motion (framer-motion)
- **API**: GitHub REST API

## 📦 Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/alvaroalof/portfolio-v2.git
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Construye para producción:
   ```bash
   npm run build
   ```

## 📄 Estructura del Proyecto

- `/src/components`: Componentes modulares de la interfaz (Hero, Skills, Projects, etc.).
- `/src/App.tsx`: Punto de entrada principal y estructura de navegación.
- `/src/index.css`: Configuración global de estilos y temas de Tailwind.
